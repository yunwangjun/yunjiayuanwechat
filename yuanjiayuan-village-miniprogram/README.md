# 贠家塬村微信小程序

## 项目介绍

这是一个为贠家塬村开发的微信小程序，旨在展示村庄的历史文化、自然风光、旅游服务和联系方式，方便村民和游客了解村庄信息。

## 功能模块

1. **首页** - 展示村庄概况和主要功能入口
2. **村庄新闻** - 发布村庄最新动态和新闻
3. **村庄历史** - 介绍村庄的历史文化和传统
4. **乡村旅游** - 展示旅游景点和特色活动
5. **联系我们** - 提供联系方式和在线留言功能

## 文件结构

```

│   ├── news/       # 新闻页面
│   ├── history/    # 历史页面
│   ├── tourism/    # 旅游页面

│   └── logs/       # 日志页面
└── utils/          # 工具函数目录
```

## 开发说明

1. 使用微信小程序原生开发
2. 遵循微信小程序开发规范
3. 采用WXML、WXSS、JavaScript技术栈
4. 响应式设计，适配不同屏幕尺寸

## 使用方法

1. 下载并安装微信开发者工具
2. 导入本项目
3. 点击预览或编译运行
